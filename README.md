# Airline Data Analytics Project

This project explores airline data to understand passenger patterns, airport activity, and operational performance.

The goal was to transform raw operational data into insights that can improve efficiency and planning.

---

## Business Problem

Airline operations generate large volumes of data, but organisations often struggle to clearly understand:

- Passenger distribution and travel patterns  
- Airport activity levels  
- Flight performance and delays  

Without clear insights, it becomes difficult to optimise operations.

---

## Approach

I used SQL to analyse passenger and flight data, focusing on key operational metrics.

The results were visualised in Power BI to create dashboards that provide a clear view of airline performance.

---

## Key Insights

- A small number of airports act as major operational hubs  
- Passenger distribution reveals key travel patterns  
- Flight status data highlights operational performance trends
